##📌 Sobre o Projeto

Este projeto foi desenvolvido utilizando o Expo, um framework que facilita a criação de aplicativos com React Native, sem a necessidade de configurar Android Studio ou Xcode.

Com o Expo, é possível criar e testar aplicativos diretamente no celular de forma simples e rápida.

##⚙️ Pré-requisitos

Antes de começar, você precisa ter instalado:

- Node.js (versão LTS)

- npm ou yarn

- Expo CLI

- Aplicativo Expo Go (disponível na Play Store e App Store)

##🚀 Como criar o projeto
>1️⃣ Instalar o Expo CLI
npm install -g expo-cli
>2️⃣ Criar o projeto
npx create-expo-app aplicativo-teste

Escolha um template:

- blank — projeto básico

- blank (TypeScript) — projeto básico com TypeScript

- tabs (TypeScript) — projeto com navegação em abas

>3️⃣ Acessar a pasta do projeto
cd aplicativo-teste
>4️⃣ Iniciar o projeto
expo start.
